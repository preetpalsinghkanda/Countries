# Where in the world?

a responsive react web application that allow user to explore countries. Users can search for countries, filter them by region and there is toggle button, switch between light theme and dark theme and view more detailed information about each country, including it's border countries.


## Demo URL

```https://countriesinfoooo.netlify.app/```



## Features 

- Dark/Light theme 
- navigate to border countries
- filter by region
- fully responsive (mobile)
- view full info about each country
- search country by their name
- display country's flag


## Tech Stack

- React JS
- Tailwind CSS 
- React Router DOM 
- Font Awesome
- Headless UI
- Google Font
